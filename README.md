# Today's Date & Time Display

A simple static HTML page showing today's date and current time across all US timezones.

## Features

- **Date Display**: Shows Thursday, May 14, 2026
- **US Timezones**: Displays current time for all 6 US timezones:
  - Eastern Time
  - Central Time
  - Mountain Time
  - Pacific Time
  - Alaska Time
  - Hawaii Time

## Deployment to GitHub Pages

1. **Create a new repository** on GitHub.com
2. **Upload the `index.html` file** to your repository
3. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"
4. **Your page will be live** at: `https://yourusername.github.io/repository-name/`

## Customization

To update the date and times, simply edit the HTML content in `index.html`. The page uses static content, so it won't update automatically.

## Technologies Used

- HTML5
- CSS3 (with gradients and responsive design)
- No JavaScript required (static page)
